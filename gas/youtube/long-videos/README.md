## Files Explained

### 1. script.html
- **Purpose**: Contains the complete video script with detailed narration
- **Format**: Standalone HTML with styling
- **Usage**: Open in browser to read the script while recording
- **Content**:
  - YouTube description
  - Detailed script with timing marks
  - Demo instructions
  - Action notes in italics

### 2. thumbnail.html
- **Purpose**: Visual thumbnail design for YouTube
- **Format**: Standalone HTML with 1280x720 canvas
- **Usage**:
  - Open in browser
  - Screenshot at full size (Cmd+Shift+4 on Mac)
  - Or use DevTools "Capture node screenshot" for exact size
- **Design**: Professional gradient backgrounds with Capital Friends branding

### 3. slides.html (Tutorial 01 only)
- **Purpose**: Interactive slideshow presentation with 15 slides
- **Format**: Full-screen HTML presentation
- **Usage**:
  - Open in browser, press F for fullscreen
  - Arrow keys or Space to navigate
  - Show during video recording as visual aid
- **Content**: Introduction, problem statement, solution, features, CTA

## How to Use These Files

### Recording a Video

1. Open `script.html` in one browser window (your reference)
2. Open your Google Sheet demo in another window
3. Follow the script step-by-step
4. For Tutorial 01, open `slides.html` in fullscreen mode and record screen

### Creating Thumbnails

1. Open `thumbnail.html` in browser
2. Use one of these methods:
   - **Screenshot**: Cmd+Shift+4 (Mac) and drag to capture
   - **DevTools**: Right-click thumbnail → Inspect → Right-click `.thumb` element → "Capture node screenshot"
3. Upload to YouTube as video thumbnail
