BEST PRACTICES

1. Create separate portfolios for separate goals — one for retirement, one for education, one for emergency. This makes goal tracking much easier later.

2. Set the rebalance threshold to 5% — this is a good default. Too low (1-2%) means constant rebalancing. Too high (10%+) means you miss imbalances.

3. Enter your correct original investment — if you know it. This ensures P&L is accurate from day one.

4. Target allocation should add up to 100% — if you have 3 funds, maybe 40-30-30. Not 50-50-50.

5. Don't create too many portfolios — one per goal is enough. Keep it simple.
